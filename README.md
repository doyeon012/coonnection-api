# Coonnection API

> 크래프톤 정글 5기 나만의 무기 프로젝트

관심사 기반으로 매칭된 사용자들이 마스크를 쓰고 비디오 채팅을 통해 대화할 수 있는 웹 서비스의 API 서버입니다.

## 기술 스택

| 분류 | 기술 |
|:---|:---|
| Runtime | Node.js |
| Language | JavaScript |
| Database | MongoDB, Redis |
| Storage | Amazon S3 |
| Auth | JWT |

## 주요 기능

- 회원가입 / 로그인 (JWT 인증)
- 관심사 기반 사용자 매칭
- 채팅 / 비디오 채팅 세션 관리
- 사용자 프로필 및 관심사 CRUD

## 실행 방법

```bash
npm install
npm run dev
```
